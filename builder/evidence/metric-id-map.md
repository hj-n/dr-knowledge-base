# Metric ID Map (Internal)

This file maps implementation IDs to user-facing metric names.
Do not expose IDs directly in end-user explanations unless the user explicitly asks for API IDs.

- `tnc`: Trustworthiness and Continuity
- `mrre`: Mean Relative Rank Error
- `lcmc`: Local Continuity Meta-Criterion
- `nh`: Neighborhood Hit
- `ca_tnc`: Class-Aware Trustworthiness and Continuity
- `l_tnc`: Label-Trustworthiness and Continuity
- `nd`: Neighbor Dissimilarity
- `snc`: Steadiness and Cohesiveness
- `dsc`: Distance Consistency
- `ivm`: Internal Validation Measure
- `c_evm`: Clustering-plus-External Validation Measure
- `stress`: Stress
- `kl_div`: Kullback-Leibler Divergence
- `dtm`: Distance-to-Measure
- `topo`: Topographic Product
- `pr`: Pearson Correlation
- `srho`: Spearman Rank Correlation
- `proc`: Procrustes Measure
- `sn_stress`: Scale-Normalized Stress
- `nm_stress`: Normalized Metric Stress
- `qnx`: QNX
- `spectral_overlap`: Spectral Overlap
